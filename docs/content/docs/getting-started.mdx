## Requirements

- Node.js `>=24`
- pnpm, pinned in the root `packageManager` field

## Install

```bash
pnpm install
```

## Run The Deck

```bash
pnpm dev
```

Open `http://127.0.0.1:5173`. Hosted docs can point the Deck link at another URL with `NEXT_PUBLIC_SLIDES_DECK_URL`.

## Run The Docs

```bash
pnpm docs:dev
```

Open `http://127.0.0.1:3000`.

## Validate Before Shipping

Optional local hook setup for this clone:

```bash
pnpm hooks:install
```

The hook runs the focused local gate:

```bash
pnpm precommit
```

For small content-only edits, run focused checks:

```bash
pnpm typecheck
pnpm lint
pnpm validate:slides
```

Before broad slide, plugin, docs, or export workflow changes ship, run the full relevant suite:

```bash
pnpm validate
```

`pnpm validate` includes dev-server and production-preview E2E smoke tests. Use manual inspection or export-specific checks when final PDF/PPTX fidelity matters.

For CI parity, run:

```bash
pnpm run ci
```

That adds the moderate dependency audit to the full validation suite. If you have Task installed, `task --list` shows aliases for the same pnpm scripts.
